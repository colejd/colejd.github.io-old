Jon Trombone

This is what happens when I have too much free time. This is a 3D scan of me hooked up to a modified version of Pink Trombone, which simulates the nose and throat to produce synthesized human vocals. You can play around with the controls in the upper-right.

Regrettably, this program supports MIDI playback. Drag any MIDI file onto my face to load it, then use the GUI to start the playback. Keep in mind that as of right now (wink wink nudge nudge) only one track and one voice is played, so chords won't work.

The source is available on GitHub.

Thanks to Toni Kaplan for repairing and rigging the 3D model.
