Reaction-Diffusion Simulator

Made with three.js.

This utilizes the Gray-Scott model to visualize the reaction and diffusion of two chemicals in in 2D. You can mess with the options in the upper-right.
See the code on GitHub.

Note: This only works on iOS with half the usual precision, so some of these configurations will not behave in the same way as they do on a desktop platform.
