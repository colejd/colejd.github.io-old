Guify

Guify is a GUI system you can use in your JavaScript projects to modify variables and trigger actions. I initially used dat.GUI while working on the other JavaScript projects on this website, but I was frustrated with its limitations and wanted something better.

Guify is my solution. It's easy to use on mobile, easy to add new functionality to, easy to customize its look, and built with accessibility in mind.

You can find more information on the GitHub repo, and you can get the latest version yourself at unpkg.com/guify or as an NPM package.
