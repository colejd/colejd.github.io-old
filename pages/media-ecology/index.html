Media Ecology Project - Waldorf

Waldorf is a system for creating and viewing rich crowd-sourced annotations on video around the Web. I did this work at the VEMILab for the Media Ecology Project at Dartmouth University. This work was sponsored under an NEH Grant.

The source is available on GitHub.
