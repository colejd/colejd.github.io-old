Dust

Epilepsy Warning: Some settings can cause bright flashing.

Dust is a WebGL-based simulator / renderer for various Cellular Automata. The simulation is largely based on CellAuto, and many of the automata came from there (see the repo for this project on GitHub for more information). My goals with this are to add interactivity and graphical features as time goes on.

Planned features:

  • Brushes

  • Creatures (automata with lifetimes and behaviors)

  • Global Line Radiosity

  • More automata
